# Lecture 10 Demo — Build vs Buy Just Changed

**Time:** ~10 minutes | **Tool:** Claude Code

## What This Proves

AI has fundamentally changed the build-vs-buy equation. Tools that once required a team and months to build can now be prototyped in hours.

## Run the Demo
