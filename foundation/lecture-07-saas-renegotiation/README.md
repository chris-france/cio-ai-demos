# Lecture 6 Demo — Renegotiating Your SaaS Stack

**Time:** ~8 minutes | **Tool:** Claude Code

## What This Proves

AI built a SaaS audit tool that shows exactly where you're overpaying — and it cost nothing to build.

## Run the Demo

Open any terminal. Type `claude`. Paste this:

> Find the cio-ai-demos repo on my machine. Inside it, go to foundation/lecture-06-saas-renegotiation. Read saas-inventory.csv. Build a complete SaaS audit tool:
> 1. Calculate cost per user per month for every tool
> 2. Flag tools over $50/user/month as 'overpriced'
> 3. Flag tools with under 40% utilization as 'underutilized'
> 4. Detect overlapping tools (e.g., multiple project management or messaging tools)
> 5. Suggest open-source alternatives for the top 5 most expensive tools
> 6. Calculate total potential annual savings
> 7. Generate a professional saas-audit.html dashboard in that folder with charts and a summary table
> Use pandas for analysis and matplotlib for charts — install if needed. Save charts as embedded base64 in the HTML. Open it in my browser when done.
