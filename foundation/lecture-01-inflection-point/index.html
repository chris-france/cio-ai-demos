Acronym Glossary

RPO — Recovery Point Objective: maximum tolerable data loss measured in time
RTO — Recovery Time Objective: maximum tolerable downtime before restoration
SOX — Sarbanes-Oxley Act: U.S. financial reporting & internal controls law
HIPAA — Health Insurance Portability and Accountability Act: U.S. healthcare data protection
PCI-DSS — Payment Card Industry Data Security Standard: credit card data protection
GDPR — General Data Protection Regulation: EU personal data privacy law
SOC 2 — System and Organization Controls 2: trust services criteria for service organizations
SAP — Systems, Applications & Products in Data Processing
ERP — Enterprise Resource Planning: integrated business process management software
CRM — Customer Relationship Management
HCM — Human Capital Management: workforce and HR systems
AD — Active Directory: Microsoft identity and access management service
ADLS — Azure Data Lake Storage: Microsoft cloud-scale analytics data store
GRS — Geo-Redundant Storage: cross-region replication for disaster recovery
WAL — Write-Ahead Log: transaction log for point-in-time database recovery
PIT — Point-in-Time: ability to restore data to any specific moment
ITSM — IT Service Management: practices for delivering IT services
ETL — Extract, Transform, Load: data integration process
DR — Disaster Recovery: strategy for restoring systems after catastrophic failure
SLA — Service Level Agreement: contractual performance commitment
KPI — Key Performance Indicator: measurable value tracking business objectives
GB / TB — Gigabyte / Terabyte: units of digital storage (1 TB = 1,024 GB)
DBA — Database Administrator: person responsible for database operations
API — Application Programming Interface: software communication protocol
M365 — Microsoft 365: cloud-based productivity suite (email, SharePoint, etc.)
S/4HANA — SAP's next-gen ERP suite running on HANA in-memory database
UTC — Coordinated Universal Time: global time standard for scheduling
