# CIO AI Curriculum — Live Demos

Demo code for the **CIO in the AI World** Udemy course by Chris France.

Each demo is a self-contained project you can clone and run locally. No proprietary data — everything uses public datasets and open-source tools.

## Structure

```
foundation/          Demos for Lectures 1-6 (Foundation Course)
advanced/            Demos for Lectures 7-12 (Advanced Course)
```

## Getting Started

```bash
git clone https://github.com/chris-france/cio-ai-demos.git
cd cio-ai-demos
```

Individual demo setup instructions are in each demo's subfolder.
